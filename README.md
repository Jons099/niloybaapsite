# Luxe Attire - Premium Women's Clothing eCommerce Platform

A full-featured, production-ready eCommerce platform built with modern technologies for selling premium women's clothing.

## 🌟 Features

### Customer Features
- 👗 Browse products with advanced filtering
- 🛒 Shopping cart with real-time updates
- 👤 Guest checkout or account creation
- 💳 Multiple payment options (COD, Card, UPI)
- 📱 Fully responsive design
- 🔍 Search and filter products
- 📦 Order tracking

### Admin Dashboard
- 📊 Real-time analytics and reports
- 👥 Employee management
- 📦 Inventory management
- 💰 Profit calculation
- 📈 Sales tracking
- 🏷️ Product management
- 📋 Order processing

### Employee Features
- ✅ Order assignment and processing
- 📊 Personal performance tracking
- 🔄 Update order status
- 📝 View assigned orders

## 🛠️ Tech Stack

### Backend
- **Runtime**: Node.js 18+
- **Framework**: Express.js
- **Database**: PostgreSQL 15
- **Cache**: Redis
- **Auth**: JWT
- **File Upload**: Multer + Sharp

### Frontend
- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Animations**: Framer Motion
- **Forms**: React Hook Form + Zod
- **Charts**: Recharts

### DevOps
- **Container**: Docker
- **Process Manager**: PM2
- **Reverse Proxy**: Nginx
- **SSL**: Let's Encrypt
